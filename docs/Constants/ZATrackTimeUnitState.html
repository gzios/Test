ZATrackTimeUnitState

@abstract TrackTimer 接口的时间单位。调用该接口时，传入时间单位，可以设置 event_duration 属性的时间单位。

@discuss 时间单位有以下选项： ZATimeUnitStateMilliseconds - 毫秒 ZATimeUnitStateSeconds - 秒 ZATimeUnitStateMinutes - 分钟 ZATimeUnitStateHours - 小时

typedef NS_ENUM(NSInteger, ZATrackTimeUnitState ) {
